✅ O que é CSS?

CSS (Cascading Style Sheets) é a linguagem usada para estilizar páginas da web. Ele define como os elementos HTML devem ser exibidos, controlando cores, fontes, espaçamento, layout e muito mais.

Na prática:

Propriedades de Texto:

Código Função Valores
font-family Permite especificar o tipo de fonte utilizada Fontes: Arial, Arial Black, Comic Sans, Courrier, New, Georgia, Impact, Times New Roman, Verdana
font-size Permite especificar o tamanho de fonte utilizada
font-weight Permite controlar a quantidade de negrito no texto Possíveis valores: lighter, normal, bold, bolder, 100, 200, 300 ... 900
font-style Permite especificar um estilo para o texto Possíveis valores: normal, italic, oblique
color Permite especificar a cor do texto
text-decoration Permite especificar uma decoração para o texto Possíveis valores: none, underline, overline, line- through (Obs: Mais de 1 valor pode ser usado)
text-transform Controla efeitos de capitalização do texto Possíveis valores: uppercase, lowercase, capitalize
text-align Controla o posicionamento horizontal do conteúdo Possíveis valores: left, right, center, justify
text-indent Define a identação para a primeira linha de um texto contida em um bloco
line-height Permite especificar a altura de uma linha de texto
letter-spacing Permite especificar o espaçamento entre letras de texto
word-spacing Permite especificar o espaçamento entre palavras do texto

Propriedade de Listas:

Código Função
list-style-type define a aparência de um marcador de uma lista
list-style-image define uma imagem como marcador de uma lista
list-style-type (listas ordenadas) Valores:decimal, lower-roman, upper-roman, georgian, armenian, none
list-style-type (não-ordenadas) Valores:disc,circle, square, none

Propriedade de Bordas:

Código Função
border-width Permite especificar a largura da borda Valores em px, porcentagem, thin, medium, thick, etc...
border-style Permite especificar um estilo para borda Valores possíveis: none, dotted, dashed, solid, groove, double, rigde
border-color Permite especificar uma cor para borda
